# Karne Abhishek Portfolio

A clean one-page portfolio website built with HTML and CSS.

## Overview

This repository contains a personal portfolio landing page for Karne Abhishek, a Full-Stack Developer and AI Automation Engineer. The site includes:

- Hero section with summary and contact CTA
- About section with professional summary and work overview
- Projects section highlighting key work
- Skills section with technical categories
- Contact section with email, LinkedIn, and GitHub links
- Dark mode support via CSS

## Files

- `index.html` — single-page portfolio website with inline CSS and layout
- `.agent.md` — workspace agent customization for portfolio editing tasks

## How to use

1. Open `index.html` in a browser.
2. Or serve the folder using a local web server, for example:

```bash
cd D:/Desktop/portfolio
python -m http.server 5500
```

Then visit `http://localhost:5500`.

## Notes

- The site is a static HTML/CSS project and does not require a build step.
- Update `index.html` directly to change content, text, or styling.
